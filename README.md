# TURNX AI Director V4

TURNX AI Director V4 is a production-ready AI creative director built with Flask and Telegram Webhooks.

It generates premium AI prompts for:
- Images
- Videos
- Image → Video

Powered by Groq API.

---

## 🚀 Features

### 3 Core Modes
- 🖼 Images
- 🎬 Video
- 🎞 Image → Video

---

## 🧠 Supported Models

### Images
- Nano Banana Pro 2
- Nano Banana Pro
- ChatGPT Image 2.0

### Video
- Veo 3.1
- Omni
- Seedance
- Grok

### Image → Video
- Veo 3.1
- Omni
- Seedance
- Grok

---

## ⚙️ Architecture

- Flask Webhook Server
- Modular Telegram Handler
- AI Director Engine (Groq-powered)
- Prompt Engine Layer
- Session State Manager
- Premium Access System
- Lightweight SQLite Logging
